# System Requirements

## Hardware
*   **OS**: Windows 10 Pro/Enterprise or Windows 11 (WSL2 supported).
*   **RAM**: Minimum 16GB (8GB for Docker, Rest for Windows). 32GB Recommended.
*   **CPU**: 4 Cores+ (Virtualization Enabled in BIOS).
*   **Disk**: 20GB free space for container images.

## Software
*   **Docker Desktop**: Latest version with **WSL2 Backend** enabled.
*   **Kind (Kubernetes in Docker)**: v0.20.0+.
*   **Kubectl**: Matching client version.
*   **Dynatrace Account**: Free trial or Enterprise tenant access.
*   **Git Bash / PowerShell**: For running commands.
